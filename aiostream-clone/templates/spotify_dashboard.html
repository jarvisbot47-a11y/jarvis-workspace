🔑 Spotify API Setup Required

  1. Go to developer.spotify.com/dashboard
  2. Create a new App (give it any name, e.g. "Mystik Stats")
  3. Copy your Client ID and Client Secret
  4. In your Spotify app settings → Redirect URIs → add:
    http://localhost:5001/spotify/callback
  5. Set environment variables and restart the server: 
    export SPOTIFY_CLIENT_ID="your_client_id"
export SPOTIFY_CLIENT_SECRET="your_client_secret"
export SPOTIFY_REDIRECT_URI="http://localhost:5001/spotify/callback"
  6. Restart the app, then visit /spotify/login

Note: Your Spotify stats require a Spotify for Artists account for full real-time data. Public API gives: followers, popularity, top tracks. Private API (if available): real-time listeners, saves, geo-data, demographics.
